Images

Insert an image using the figure tags with a caption. You can set the image width and height directly in the img tag or in the bootstrap-addon css file by assigning the image an id and styling the id.

Insert an image here with a hyperlink. Put the image tag inside the anchor tags.

Insert an image here and create a float by using the pull-left or pull-right class. This will make the text wrap around it. This is text after the image. You can add a border around the image using the img-thumbnail class. You can make rounded corners by adding the img-rounded class. You can make it completely round by adding the img-round class. To adjust the space around the image you will need to add padding or margin. To do that you'll need to insert an id attribute then add the padding or margin to your addin stylesheet.

It is best to save the image to the size you want to display it, then include the actual width and height in the style rule. You can find out the dimensions of an image by right clicking on it and looking in the properties.

To stop wrapping elements use the clearfix class. Without this the next header section would wrap around the image.

Glyphicons

Glyphicons are a set of 200 icons created by Glyphicon.com and bundled free with Bootstrap. They are saved locally in your fonts folder. They can be used in your website to provide visual cues such as the home icon next to the home link in your navigation bar. We are using them in the sidebar of this site.

Buttons

Button Classes:

Use the HTML button element to create a button. Use the type="button" attribute to ensure consistent presentation across browsers. For Bootstrap styling and color add the btn and btn-style class.

Button Sizes:

Add btn-size class to the opening button tag to change the size of the button

add btn-block class to the opening button tag to make it block level.

Use the button classes with hyperlinks and Submit buttons:

Insert the btn btn-style class in the opening tag of the anchor (i.e., hyperlink) element to make it appear as a button.

Link

You can use the same method when creating a form submit or reset button. Insert the btn btn-style class in the opening tag of the input element. More on this in the Forms section.
